Evolución Web

Web es un vocablo inglés que significa “red”, “telaraña” o “malla”. El concepto se utiliza en el ámbito tecnológico para nombrar a una red informática y, en general, a Internet. Su historia es:

  • Era Militar
  • Ordenador Personal
  • WEB 1.0
  • WEB 2.0
  • WEB 3.0
  • WEB 4.0
